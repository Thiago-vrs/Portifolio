Projeto 2 - Automatizando Dados do Google Sheets e Importando para o Excel

Desenvolvi um script em Python que automatiza a extração de dados do Google Sheets, corrige inconsistências e exporta os dados tratados para o Excel. O código:

  • Remove valores nulos ou preenche com dados padrão.
  • Corrige tipos de dados para garantir compatibilidade.
  • Remove duplicatas para evitar redundâncias.
  • Padroniza formatação de texto para melhor visualização.
Esse processo melhora a integridade dos dados e facilita sua análise no Excel.
